Mobile Responsiveness Features

  • Touch Targets: All buttons and controls have 44px minimum height for easy mobile tapping
  • Smart Header: On mobile, header stacks vertically to save space; stats moved to progress bar
  • Full-Width Controls: Dropdowns and buttons expand to fill width on phones for easier selection
  • Optimized Sizing: Mission rows have 56px minimum height; checkboxes enlarged to 16px
  • Responsive Proportions: Map gets 50vh on phones (priority), panel gets 40vh
  • Text Scalability: Font sizes increase on mobile (13px buttons vs 11px desktop)
  • WCAG Compliant: Meets accessibility guidelines for color contrast and touch target sizes
  • Flexible Wrapping: Header controls wrap and flex to available space
  • Three Breakpoints: Optimized layouts for 480px (mobile), 768px (tablet), and 1200px+ (desktop)
  • Touch-Friendly Spacing: Increased padding and margins for easier scrolling and tapping
